# Delivery Management System - Master Database Schema

This schema is optimized for **PostgreSQL** and **Laravel Eloquent**, utilizing PostGIS for spatial data and strict decimal typing for financial accuracy.

---

## Domain 1: Users, Roles & Fleet Management

### 1. `users`
Core identity table for all actors in the system.
| Column | Type | Constraints / Notes |
| :--- | :--- | :--- |
| `id` | BigInt | Primary Key, Auto-increment |
| `role` | Enum | 'admin', 'dispatcher', 'hub_operator', 'driver' (Indexed) |
| `name` | String | |
| `phone` | String | Unique, Indexed (Primary login for drivers) |
| `email` | String | Nullable |
| `password` | String | Hashed |
| `base_location_id` | BigInt | Foreign Key (`locations.id`), Nullable |
| `status` | Enum | 'active', 'suspended', 'inactive' |
| `timestamps` | Timestamp | `created_at`, `updated_at`, `deleted_at` |

### 2. `vehicles`
Physical assets assigned to drivers.
| Column | Type | Constraints / Notes |
| :--- | :--- | :--- |
| `id` | BigInt | Primary Key |
| `driver_id` | BigInt | Foreign Key (`users.id`), Unique (One-to-One) |
| `type` | Enum | 'motorcycle', 'tuktuk', 'minivan', 'box_truck' |
| `plate_number` | String | Unique |
| `max_weight_kg` | Decimal(8,2) | Limits routing assignments |
| `max_volume_cbm`| Decimal(8,2) | Limits routing assignments |
| `status` | Enum | 'active', 'maintenance', 'retired' |

### 3. `driver_metrics`
Rolling performance data for dispatching logic.
| Column | Type | Constraints / Notes |
| :--- | :--- | :--- |
| `driver_id` | BigInt | Primary/Foreign Key (`users.id`) |
| `current_rating`| Decimal(3,2) | Rolling average (last 100 deliveries) |
| `total_deliveries`| Integer | Lifetime count |
| `completion_rate`| Decimal(5,2) | Percentage |
| `on_time_percent`| Decimal(5,2) | Percentage |
| `total_cod_lifetime`| Decimal(15,2) | Lifetime COD collected |

| `weight_kg` | Decimal(6,2) | |
| `volume_cbm` | Decimal(6,2) | |
| `recipient_name` | String | |

| `dropoff_address` | Text | |

## Domain 5: Tracking, Telemetry & Exceptions

### 10. `tracking_logs`

| Column | Type | Constraints / Notes |
| :--- | :--- | :--- |
| `id` | BigInt | Primary Key |
| `order_id` | BigInt | Foreign Key (`orders.id`) |

| `lat`, `lng` | Decimal | Nullable (If scanned outside hub) |
| `note` | String | Nullable |
| `created_at` | Timestamp | |

### 11. `driver_telemetry`

| Column | Type | Constraints / Notes |
| :--- | :--- | :--- |

| `driver_id` | BigInt | Foreign Key (`users.id`) |
| `route_id` | BigInt | Foreign Key (`routes.id`) |
| `location` | Geometry(Point) | PostGIS |
| `speed_kmh` | Integer | |
| `heading` | Integer | 0-360 degrees (for Mapcn rotation) |
| `created_at` | Timestamp | |

### 12. `delivery_issues`

| Column | Type | Constraints / Notes |
| :--- | :--- | :--- |

| `driver_id` | BigInt | Foreign Key (`users.id`) |
| `issue_type` | Enum | 'severe_weather', 'road_closed', 'vehicle_breakdown', 'traffic_accident', 'system_app_failure', 'customer_unreachable', 'customer_requested_reschedule', 'access_denied', 'safety_hazard', 'package_missing', 'package_damaged' |
| `photo_url` | String | S3 Link, Nullable |
| `status` | Enum | 'open', 'investigating', 'resolved' |
| `timestamps` | Timestamp | `created_at`, `updated_at` |

### 13. `reviews`
Post-delivery customer ratings.
| Column | Type | Constraints / Notes |
| :--- | :--- | :--- |
| `id` | BigInt | Primary Key |
| `order_id` | BigInt | Foreign Key (`orders.id`) |
| `driver_id` | BigInt | Foreign Key (`users.id`) |
| `rating` | Integer | 1 to 5 |
| `comment` | Text | Nullable |
| `is_exempt` | Boolean | Default: False (Admin override) |
| `created_at` | Timestamp | |
